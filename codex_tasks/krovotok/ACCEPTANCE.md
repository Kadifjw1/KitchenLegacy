# Кровоток — критерии приёмки

Задача считается выполненной, когда соблюдены все пункты:

- [ ] `worldsmith:krovotok` зарегистрирован и доступен в творческой вкладке Worldsmith.
- [ ] Название отображается как «Кровоток» в `ru_ru` и “Bloodflow” в `en_us`.
- [ ] В игре используется исходная геометрия из 170 элементов без изменения силуэта.
- [ ] Основная текстура багрового канала анимируется через `.mcmeta`.
- [ ] Отображаются шесть визуальных состояний заряда 0–5.
- [ ] Заряд хранится в NBT ItemStack и не переносится между разными экземплярами меча.
- [ ] Серия сбрасывается после перерыва более 60 тиков.
- [ ] На 1–5 зарядах применяются бонусы урона и скорости атаки без дублирующихся attribute modifiers.
- [ ] Удар после достижения 5 зарядов выполняет разрядку, наносит дополнительный урон, лечит владельца и сбрасывает заряд.
- [ ] Зарегистрированы и работают пять типов частиц Кровотока.
- [ ] Пассивные частицы не создают заметной нагрузки и не спамят мир.
- [ ] Life-drain частицы визуально идут от цели к игроку.
- [ ] Новые Java-классы используют package `ru.theframetrip.worldsmith`.
- [ ] Все новые ресурсы используют namespace `worldsmith`.
- [ ] `./gradlew clean build` завершается успешно.
- [ ] Dedicated server запускается без ошибок client-only class loading.
- [ ] Существующая функциональность Предела/Разлома остаётся рабочей.
