# Worldsmith

Worldsmith is a Minecraft Forge 1.20.1 mod base for fantasy weapons, abilities, and artifacts.

Current approved work preserved in this repository:

- Codex task specifications under `codex_tasks/`.
- The sword **Предел** (`worldsmith:predel`) and its active ability **Разлом**.
- Approved Predel/Rift resources such as `predel_void.json`, `tex_predel.png`, and void particles.
